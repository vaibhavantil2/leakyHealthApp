# leakyHealthApp

A simple Android demo app written in Java that showcases basic user workflows and contains intentionally leaky / insecure code patterns for static analysis testing.

## Overview

- Java-based Android app using standard Android SDK components.
- Includes screens for login, signup, personal info, pharmacy info, and drug information.
- Designed as a lightweight example project for security analysis and code scanning.

## Project structure

- `app/` - Android application module.
- `app/src/main/java/io/privado/privadohealthapp/` - app source code.
- `app/src/main/res/` - layouts, resources, and themes.
- `app/src/androidTest/` and `app/src/test/` - instrumentation and unit tests.

## How to build

1. Open the project in Android Studio.
2. Sync Gradle.
3. Run the app on an emulator or device.
